# StockTrade Pro

**StockTrade Pro** is a responsive stock trading simulator built with HTML, CSS, and JavaScript. The project recreates a modern trading terminal experience with real-time market simulation, portfolio tracking, watchlists, alerts, admin analytics, and algorithmic trading controls in a single front-end application.

**Live Demo:** https://aksharpatel007.github.io/Stock-Trade/

## Core Features
- Real-time stock market simulation with dynamically updated prices and trend movement
- Interactive trading dashboard with chart switching, indicators, ticker tape, and quick trade dock
- Market and limit order support with stop-loss and take-profit inputs
- Portfolio tracking with allocation insights and performance monitoring
- Stock screener with search, sector, and trend filters
- Watchlist, movers board, leader ranking, and price alert management
- Trade history export to PDF for reporting and record keeping
- Admin dashboard with user analytics, market controls, and trade monitoring
- Dark mode support and responsive layout for desktop and mobile devices
- Browser-based persistence using Local Storage for accounts, sessions, and portfolio data

## Technology Stack
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
- **UI Framework:** Bootstrap 5, Bootstrap Icons
- **Charts & Visualization:** ApexCharts
- **Document Export:** jsPDF, jsPDF-AutoTable
- **Storage:** Browser Local Storage
- **Deployment:** GitHub Pages
